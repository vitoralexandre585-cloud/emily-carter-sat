# Dr. Emily Carter — SAT Preparation

Single-file responsive landing page for a premium NYC SAT tutor.

**Live site:** https://vitoralexandre585-cloud.github.io/emily-carter-sat/

## Features
- Accessible mobile navigation (hamburger with aria-expanded, Esc / link / resize close)
- Photo with an automatic "EC" monogram fallback if the image fails to load
- Keyboard focus rings and `prefers-reduced-motion` support
- No build step — open `index.html` in any browser

## Local preview
```
python -m http.server 4000
# then open http://localhost:4000
```
